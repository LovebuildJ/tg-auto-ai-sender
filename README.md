**🚀 项目概述**   
> 本项目是一个基于autojs和deepseek的智能水群工具，旨在解决无时间待需要群活跃问题。通过自动化和智能化技术方案，帮助用户实现一键挂机自动水群。

---

**🌟 功能亮点**  
✔️ 功能1 - 安装即用，无需懂任何技术
✔️ 功能2 - 对电报无入侵，绝对不会因为使用本程序而炸号  
✔️ 功能3 - 接入最强AI deepseek, 对话更加智能化，且经过模型微调已很难看出是否是ai
✔️ 功能4 - 随时随地，想挂就挂，支持双写模式(即挂机时，也可以插入自己手动输入)
